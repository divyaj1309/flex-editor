# flex React Editor - A Feature-Rich WYSIWYG Text Editor for React

**flex React Editor** is a highly customizable, feature-rich **WYSIWYG** (What You See Is What You Get) text editor for **React** applications. With comprehensive support for text formatting, tables, hyperlinks, and more, it is ideal for developers who need to integrate a robust and flexible **rich-text editor** into their web applications.

## 📦 Installation
Install **flex Editor** via npm:
```sh
npm install flex-editor-pro
```

## 🛠️ Usage

1. Import flex Editor into your React project:

    ```javascript
    import React from 'react';
    import TextEditor from 'flex-editor-pro';
    import 'flex-editor-pro/dist/TextEditor.css';
    ```

2. Use the editor in your component:

    ```javascript
    const App = () => {
      return (
        <div>
          <TextEditor />
        </div>
      );
    };

    export default App;
    ```
